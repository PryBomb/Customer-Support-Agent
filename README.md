# Customer Support Agent

A customer support AI agent developed as part of the **Kaggle Agents Course – Day 3: Agent Skills**.

This project explores the fundamentals of building AI agents capable of handling customer interactions, processing user requests, and providing intelligent responses through structured workflows.

---

## About the Project

The goal of this project is to understand and implement core agent skills, including:

- Prompt engineering
- Tool usage and function calling
- Memory and context management
- Multi-step reasoning
- Task execution workflows
- Customer support automation

This repository contains a basic customer support agent skeleton that will be expanded throughout the Kaggle course.

Installation

Clone the repository:

git clone https://github.com/YOUR_USERNAME/customer_support_agent.git
cd customer_support_agent

Create and activate a virtual environment.

Windows
python -m venv venv
venv\Scripts\activate
macOS/Linux
python3 -m venv venv
source venv/bin/activate

Install dependencies:

pip install -r requirements.txt
Running the Project

Run the test script:

python test_agent.py
